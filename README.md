## 我完成了什麼

- 建立 `index.html`：
  - 單頁主介面（Main Interface）
  - 顯示使用者名稱 **kai chi**
  - 動態問候語（早安 / 午安 / 晚安）
  - 即時顯示目前時間與日期
  - 專業區塊：About、Skills、Projects
- 建立 `style.css`：
  - 以 CSS 變數打造 Glassmorphism 設計系統
  - 漸層背景與玻璃卡片視覺
  - 微動畫（浮動、淡入、hover 效果）
  - 響應式版面（手機與桌面）
- 在 `index.html` 內加入 JavaScript：
  - 每秒更新時間
  - 依照時段更新 greeting
  - 加入卡片滑鼠互動（含 reduced motion 無障礙考量）

## 檔案結構

- `index.html`：Core structure and content
- `style.css`：Visual design and animations
- `README.md`：Project documentation

## 使用方式

1. 將本專案推送到 GitHub Repository。
2. 到 Repository 的 **Settings → Pages**。
3. Source 選擇部署分支（例如 `main`）與根目錄（`/root`）。
4. 儲存後稍等片刻，即可透過 GitHub Pages 網址查看。
